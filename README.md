# MultiRoute

A React Native transit app for New York City that supports multi-stop routing. Unlike traditional transit apps, MultiRoute lets you plan trips with customizable stops, adjust stop durations, and view routes based on real-time transit data.

## Features
- Plan trips with multiple stops.
- Adjust the duration at intermediate stops.
- View real-time transit updates for each leg of the trip.

## Stack
- React Native (Expo)
- Google Directions API / MTA GTFS
- React Navigation

## Development Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/quence-dev/MultiRoute.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   expo start
   ```

4. Test on your device with the Expo Go app or an emulator.
